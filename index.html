Learning HTML 0.1
  1. Heading, There is 6 Heading type. Example:

    1. Heading 1 <h1>

    2. Heading 2 <h2>

    3. Heading 3 <h3>

    4. Heading 4 <h4>

    5. Heading 5 <h5>
    6. Heading 6 <h6>
  2. Paragraph, A paragraph always starts on a new line, and is usually a block of text. Example :

    This paragraph contains a lot of lines in the source code, but the browser ignores it.

    This paragraph contains a lot of spaces in the source code, but the browser ignores it.

  3. Entities, some characters already resevered by HTML, so we can't use characters in some text, so we need to use symbol entities. Example :

    • for < you need to use &lt;
    • > you need to use &gt;
    • ( you need to use &lpar;
    • ) you need to use &rpar;

    we can see the list of all entities here : Entities List

  4. Break Line, you can't use enter to make a new line but you need to use break line. Example :

    1. <br>, to insert a single line break

    2. <hr>, to insert a single line break and add a horizontal rule

  5. Style, in HTML you can add atribute style to change color, font (familiy ,size and color), background color, etc. Example:
    1. font-family, to use this you can add: style="font-familiy: Courier;"
    2. background-color, to use this you can add: style="background-color: aqua;"
    3. color, to use this you can add: style="color: red;"
    4. font-size, to use this you can add: style="font-size: 200%;"

  6. Format, you can change your text format to bold, italic, etc. Example :
    1. Bold <B> and <strong>
    2. Italic <i>
    3. Small <small>
    4. Strike through <del>
    5. Underline <ins>
    6. Subscript <sub>
    7. Superscript <sup>
  7. Comment, can be used to add information for developer that not showed in the web page. Example:

    <!-- Comment -->

  8. List, to make a list you can use <ul> (unorderer list) or <ol> (ordered list), for the text you want to list you use <li> (list item). Example:

    example for <ul> :

    My hobby:

    • Gaming
    • Mewing
    • Playing

    example for <ol>:

    My hobby:

    1. Gaming
    2. Mewing
    3. Playing

  9. Link, in HTML we can embed our link. Example:
    1. target="_self" to open in the same page: How to Sigma
    2. target="_blank" to open in new page
